1> Java中只能单继承。
2> Sub(子类)会继承Super(父类)的所有属性，包括private修饰的。
3> Java继承中方法的重写要遵循：两同两小一大。
	1) 方法名和传入的参数必须相同
	2) 子类方法返回值和异常必须比父类方法小
	3) 子类方法的访问权限要比父类大
4> Sub(子类)构造器中必须首先调用Super(父类)构造器。
	如果子类构造器中没有调用父类构造器，则程序会自动调用父类的无参构造器。
	如果父类没有无参构造器，则必须写出调用的构造器。
5> Super(父类)构造器中调用被Sub(子类)重写的方法时，调用的是重写后的方法。
6> Java实例化子类，执行顺序：
	sup static block --> sub static block --> sup block -->
	sup constructor -- > sub block --> sub constructor
	
Java继承中方法的重写要遵循：两同两小一大。
	两同：方法名和传入的参数必须相同；
	两小：子类方法返回值和异常必须比父类方法小
	一大：子类方法的访问权限要比父类大

子类方法抛出异常需遵循：
	可以	
		1：子类重写父类方法时，可以不抛出任何异常
		2：重写时可以仅抛出父类的部分异常
		3：重写时抛出父类抛出异常的子类异常
	但不能：
		1：抛出额外的异常（父类没有抛出的）
		2：抛出父类方法抛出的异常的父类异常
